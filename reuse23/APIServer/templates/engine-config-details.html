Engines & Configurations

Hover to learn about the different engines and their different configurations.

JavaScriptCore

JavaScriptCore (jsc) is Apple's web engine for the Safari web browser. We used three tiering configurations for testing:

  • jsc-int: JITs disabled and rewriting interpreter enabled
  • jsc-bbq: interpreter enabled and only BBQ quick JIT enabled
  • jsc-omg: interpreter enabled and only OMG optimizing JIT enabled

V8

V8 is Google's web engine for the Chrome web browser. We used two tiering configurations for testing:

  • v8-liftoff: only Liftoff baseline JIT enabled
  • v8-turbonfan: only TurboFan optimizing JIT enabled

SpiderMonkey

SpiderMonkey (sm) is Mozilla's web engine for the Firefox web browser. We used two tiering configurations for testing:

  • sm-base: only baseline compiler enabled
  • sm-opt: only optimizing compiler enabled

wasmtime

wasmtime is a non-web WebAssembly engine written in Rust and uses the cranelift optimizing compiler. We tested on wasmtime's default configuration.

Wasm3

Wasm3 is a WebAssembly interpreter and a non-web engine used for embedded systems. We tested with Wasm3's default configuration.

WAVM

WAVM is a non-web engine that uses LLVM to compile WebAssembly code ahead-of-time. We tested with WAVM's default configuration.

wazero

wazero is a non-web WebAssembly engine written in Go. We tested on wazero's default configuration.

Wizard

Wizard (wizeng) is a WebAssembly research engine developed by Ben Titzer. We used two tiering configurations for testing:

  • wizeng-int: only interpreter enabled
  • wizeng-jit: only JIT compiler enabled

iwasm

iwasm is WebAssembly Micro Runtime's non-web engine mini product. We used two tiering configurations for testing:

  • iwasm-int: only interpreter enabled
  • iwasm-fjit: only fast JIT enabled

Wasmer

Wasmer is a WebAssembly non-web engine written in Rust and uses the Cranelift optimizing compiler. We used two tiering configurations for testing:

  • wasmer: default configuration with Cranelift
  • wasmer-base: single-pass compiler enabled
